# Changelog

All notable changes to AI Slidebar will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-02-05

### Added
- Initial public release
- Auto-hiding sidebar triggered by cursor at screen edge
- Support for 12 AI services:
  - ChatGPT, Claude, Gemini, Perplexity, Grok, Pi
  - HuggingChat, Mistral, Poe, Copilot, Claude Code, You.com
- Multi-monitor support with dynamic detection
- Configurable sidebar position (left/right side)
- Quick Prompts system with fast-access toggles
- Prompt injection into AI chat inputs
- Pin mode to keep sidebar visible
- Dark/Light mode toggle
- Zoom control (50-200%)
- "Keep Chat" feature to preserve chat context
- DPI-aware rendering for high-resolution displays
- Settings persistence via JSON
- Comprehensive logging for troubleshooting

### Security
- No telemetry or data collection
- All data stored locally
- Open source for full transparency
